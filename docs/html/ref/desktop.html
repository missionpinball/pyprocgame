desktop Submodule¶

Desktop¶

class procgame.desktop.Desktop¶

The Desktop class helps manage interaction with the desktop, providing both a windowed representation of the DMD, as well as translating keyboard input into pyprocgame events.

add_key_map(key, switch_number, sticky_key=False, initial_state='open')¶

Maps the given key to switch_number, where key is one of the key constants in pygame.locals.

clear_key_map()¶

Empties the key map.

draw(frame)¶

Draw the given Frame in the window.

exit_event_type = 99¶

Event type sent when Ctrl-C is received.

get_keyboard_events()¶

Asks pygame for recent keyboard events and translates them into an array of events similar to what would be returned by :meth: pinproc.PinPROC.get_events.
