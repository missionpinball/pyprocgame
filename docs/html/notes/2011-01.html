pyprocgame 1.0 Release Notes¶

The following change notes are a summary of changes between git hash 8a769c077d1c184f5dcd and 1.0:

pyprocgame¶

  • Added dmdsplashrom procgame tool which enables requesting custom power-up DMD images.

pypinproc¶

Added new methods supporting pinballcontroller.com’s new Power Driver Boards:

  • pinproc.PinPROC.driver_update_global_config()
  • pinproc.PinPROC.driver_update_group_config()
  • pinproc.PinPROC.write_data()
